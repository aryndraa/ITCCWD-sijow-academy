Membangun Front End Website dengan Angular JS

AngularJS adalah sebuah framework JavaScript yang dikembangkan oleh Google untuk membangun aplikasi web dinamis, terutama Single Page Applications (SPA). AngularJS pertama kali dirilis pada tahun 2010 dan sejak itu menjadi salah satu framework paling populer untuk pengembangan aplikasi web. AngularJS memungkinkan developer untuk menggunakan HTML sebagai template language dan memperluas sintaks HTML untuk mendeklarasikan komponen aplikasi secara lebih jelas dan mudah dikelola.

Keunggulan Angular JS

Two-Way Data Binding

Menghemat waktu dan usaha pengembang karena perubahan pada model secara otomatis tercermin di view, dan sebaliknya, tanpa perlu kode tambahan untuk memperbarui UI

Modular dan Terstruktur

Arsitektur MVC membuat pengembangan aplikasi lebih terstruktur. Kode yang terpisah antara tampilan (view), data (model), dan logika (controller) memudahkan pemeliharaan dan pengembangan aplikasi skala besar.

Directives dan Reusability

AngularJS memungkinkan developer untuk membuat directives khusus yang dapat digunakan kembali, sehingga mempercepat proses pengembangan dan meningkatkan efisiensi.

Komunitas Besar dan Dukungan Google

Karena dikembangkan oleh Google, AngularJS mendapat dukungan yang kuat dari komunitas serta memiliki dokumentasi yang sangat lengkap. Ada banyak resource, tutorial, dan plugin yang tersedia untuk membantu pengembangan dengan AngularJS

Modul Pembelajaran

1

HTML & CSS

Angular bekerja dengan HTML dan CSS dalam templatenya. Pemahaman mendalam tentang HTML dan CSS sangat diperlukan.

  • Struktur dokumen HTML
  • Elemen-elemen dasar (div, h1, p, span, dll.)
  • Form dan input data
  • Flexbox & Grid
  • Animasi dan Transisi
  • Responsivitas (Media Query)
  • CSS Preprocessors (Sass, LESS)
2

Dasar-Dasar JavaScript

Sebelum mempelajari Angular, sangat penting untuk memiliki pemahaman yang baik tentang JavaScript, khususnya TypeScript yang menjadi dasar Angular.

  • Variabel (var, let, const)
  • Tipe Data (String, Number, Boolean, Array, Object)
  • Kontrol Alur (If, Else, Switch)
  • Loop (for, while)
  • Fungsi (Function, Arrow Functions)
  • Object dan Array Methods (map, filter, reduce)
  • DOM Manipulation (querySelector, addEventListener)
  • Asynchronous Programming (Promises, Async/Await)
  • ES6+ (Spread Operator, Destructuring, Template Literals)
3

Dasar-Dasar Angular JS

Setelah menguasai JavaScript dan TypeScript, mulailah dengan dasar-dasar Angular.

  • Pengenalan Angular Js
  • Instalasi Angular CLI
  • Membuat dan Menjalankan Proyek Angulars
  • Struktur Aplikasi Angular
  • Templating di Angular (interpolasi, binding)
  • Data Binding (One-way dan Two-way)
  • Direktif Dasar (*ngIf, *ngFor)
  • Angular Modules dan Komponen
  • Lifecycle Hooks Komponen (ngOnInit, ngOnChanges, dll.)
  • Angular Services dan Dependency Injection
4

Komponen di Angular

Komponen adalah elemen utama di Angular. Pelajari cara membuat dan mengelola komponen.

  • Membuat Komponen dengan Angular CLI
  • Data Binding di Komponen (Property Binding & Event Binding)
  • Input dan Output Decorators (Passing data antara komponen)
  • Komponen Lifecyclel
  • Komponen Dinamis dan Asynchronous
  • Mengelola Templating dengan Direktif (ngIf, ngFor, ngClass, ngStyle)
  • Child vs Parent Komponen Komunikasi
5

Routing dengan Angular Router

Mengelola navigasi halaman pada aplikasi SPA (Single Page Application) dengan Angular Router.

  • Setup dan Konfigurasi Routing
  • RouterLink dan Navigasi Programatis
  • Route Dinamis (Params)
  • Nested Routes
  • Route Guards untuk Autentikasi
  • Lazy Loading Routes untuk optimasi performa
6

Forms dan Validasi

Mengelola input dari pengguna melalui formulir dan menerapkan validasi.

  • Reactive Forms vs Template-Driven Forms
  • FormGroup, FormControl, dan FormBuilder
  • Two-way Data Binding dengan ngModel
  • Validasi Form (Built-in Validators dan Custom Validators)
  • Error Handling pada Form
7

Konsumsi API dan HTTP Client

Pelajari cara berinteraksi dengan backend menggunakan HTTP di Angular.

  • Menggunakan Angular HttpClientModule
  • GET, POST, PUT, DELETE Request
  • Handling Responses dan Error dengan Observables
  • Penggunaan RxJS (Reactive Extensions for JavaScript)
  • Menggunakan Interceptors untuk Authentication dan Authorization
8

State Management di Angular

Belajar mengelola state aplikasi dengan baik, terutama pada aplikasi yang kompleks.

  • Menggunakan Angular Service untuk mengelola state antar komponen
  • Shared Service untuk komunikasi antar komponen
  • Store dan Reducer pada NgRx
  • Actions dan Effects pada NgRx
  • Selectors untuk mendapatkan state tertentu
  • Penggunaan NgRx DevTools untuk debugging
9

Dependency Injection dan Services

Memahami konsep Dependency Injection dan bagaimana Angular mengimplementasikannya.

  • Membuat dan Menggunakan Service
  • Injecting Service ke dalam Komponen
  • Hierarki DI pada Angular (Module-level vs Component-level)
  • Injector Token dan Optional Dependency Injection
10

Testing di Angular

Mengetes aplikasi Angular untuk memastikan kualitas kode.

  • Testing Komponen dan Service dengan Jasmine dan Karma
  • Menggunakan Angular TestBed untuk menguji komponen
  • Mocking Services dalam Unit Test
  • Testing dengan Protractor atau Cypress
  • Scenario Testing pada aplikasi Angular
11

Optimasi dan Deployment

Mempelajari cara mengoptimalkan performa aplikasi Angular dan melakukan deployment.

  • Lazy Loading Modules
  • AOT (Ahead-of-Time) Compilation
  • Tree Shaking
  • Minifikasi kode dan mengelola bundle size
  • Menggunakan Angular Universal untuk Server-Side Rendering (SSR)
  • Build Aplikasi Angular (ng build)
  • Deploy ke platform seperti Netlify, Vercel, Firebase, atau AWS
  • Penggunaan Continuous Integration dan Deployment (CI/CD)
